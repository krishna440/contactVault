# ContactVault – Smart Contact Manager

ContactVault is a secure and feature-rich contact management web application built using Spring Boot. It allows users to manage contacts, upload profile images, set reminders, and securely authenticate using OAuth and Spring Security.

## Features

• User Registration and Login  
• Google OAuth Authentication  
• Add, Update, Delete Contacts  
• Upload Contact Profile Images  
• Contact Search Functionality  
• Reminder Scheduling System  
• Email Service Integration  
• Secure Authentication and Authorization  
• Responsive UI using Tailwind CSS  

## Tech Stack
